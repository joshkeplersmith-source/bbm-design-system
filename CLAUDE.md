# BBM Design System

This repo contains the Body Balance Medical brand design system — the single source of truth for all brand assets, design tokens, and visual guidelines used across web, print, and digital productions.

**Client:** Body Balance Medical  
**Managed by:** Captain Kepler Creative / Josh Smith  
**Spec:** See `BB-DESIGN-SYSTEM-SPEC.md` for full design system documentation.

## Contents

- `BB-DESIGN-SYSTEM-SPEC.md` — full brand spec (typography, color tokens, spacing, component rules)
- `BRAND.md` — brand voice and usage guidelines (in progress)
- `tokens.css` — CSS custom properties for all design tokens (in progress)
- `logos/` — all logo lockups and mark variants (SVG)
- `textures/` — brand texture assets (SVG + WebP)
- `photography/` — licensed hero and lifestyle photography (JPG)
- `references/` — rendered HTML version of the design spec
