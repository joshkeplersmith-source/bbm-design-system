How to Build with AI
1
Start a New AI Chat
Open a fresh conversation every time. Never reuse an existing chat — leftover context from other sessions affects output quality.
2
Load the Brand File
Upload BB-BRAND-TEAM.html directly to the chat, or reference it from your project files. This is the source of truth for all Body Balance brand rules.
3
Give Your Brief
Describe what you need to build. Include the deliverable type, any copy or text, layout preferences, event details, or requirements. The more specific, the better the output.
4
Build
Tell the AI to produce the deliverable and let it run. Review the output once it's complete before making any changes.
5
Verify
Send this prompt after every build:
"Verify that everything in this output can be traced back to the brand design system. Flag anything that can't."
6
Tweak
Review the output visually. If anything looks off — overlapping text, spacing issues, anything that doesn't feel right — describe it and ask for a fix.
7
Publish
Once the verify step passes and any tweaks are done, the deliverable is ready to use.
Tip
If your AI already has Body Balance project files or context from older sessions, run a quick session to update it with the new brand design system. The HTML file supersedes any previous brand information.

Always load the file The brand file must be present every time you build. It is the source of truth.
Always verify Don't skip Step 5. If anything gets flagged, ask for a fix before moving on.
